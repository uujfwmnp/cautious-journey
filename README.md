# cautious-journey
Command-line camera changing utility for iRacing, now with an optional appJar-based GUI!

### Requirements:
 - Python 3.4 or newer
 - PyYAML 3.11 or newer
 - [appJar](http://appjar.info) 0.94 or newer (only if using GUICamera.py)
 - [Python iRacing SDK](https://github.com/kutu/pyirsdk)
 
### To use:
 - Run `pip install pyirsdk` to install the Python iRacing SDK
 - Run the iRacing program
 - Run `python3 irCamera.py` to start the script

### To use the GUI:
 - Run `pip install pyirsdk` to install the Python iRacing SDK
 - Run `pip install appjar` to install appjar
 - Run the iRacing program
 - Run `python3 GUICamera.py` to start the script
